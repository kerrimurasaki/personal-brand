# Border Radius

| Token | Value | Default usage |
|---|---|---|
| base | 32px | Cards, modals, accordion, tables, alerts, large containers |
| default | 10px | Badges, tooltips, dropdown items, small controls |
| sm | 6px | Checkboxes, tiny elements |
| full | 9999px | Buttons, inputs, pills, avatars, toggles, dot indicators |

## Rules

- 9999px (pill) is the default radius for buttons and inputs
- 32px is the default radius for cards and containers
- Never use arbitrary radius values outside this scale
- Radius must be consistent within each component family
